Drop product cover images here.

The Products section looks for:
  - keeper.png   -> Keeper card cover
  - invoice.png  -> Invoice Generator card cover

Recommended: 16:10 ratio (e.g. 1280×800), PNG or JPG.
If a file is missing, the card automatically falls back to its
gradient + aperture mark, so nothing breaks in the meantime.

To use a different filename or format, update `image:` in
src/data/projects.ts (e.g. "/projects/keeper.jpg").
